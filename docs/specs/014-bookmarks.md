# Spec [014]: Personalization (Bookmarks)

**Feature ID**: `014-bookmarks`

## Overview

Allow users to save events and routes locally in their browser.

## Implementation Tasks

- [ ] **Build the localStorage Bookmark**: Build the `localStorage` Bookmark manager utility.
- [ ] **Add Bookmark toggle buttons**: Add Bookmark toggle buttons to Event and Route cards with a 'Device Only' first-time warning.
- [ ] **Create the Unified Bookmarks**: Create the Unified Bookmarks Page (`pages/bookmarks.astro`) with client-side hydration.
- [ ] **Implement Bookmark Export (JSON**: Implement Bookmark Export (JSON download) and Import (JSON parse) logic.
